# WebXR VR Prototype - SexLikeReal UI Viewer

Простой прототип для просмотра дизайна интерфейса в VR очках Meta Quest 3.

## Функции

- ✅ WebXR сессия для просмотра в VR
- ✅ Переключение между плоским и изогнутым видом
- ✅ Настраиваемые параметры: расстояние, ширина панели
- ✅ HTML/CSS интерфейс с качественным отображением
- ✅ Отладочная панель для тестирования параметров

## Требования

1. **HTTPS соединение** - WebXR работает только через HTTPS
2. **Meta Quest 3** или другие VR очки с поддержкой WebXR
3. Современный браузер (Chrome, Edge, Firefox)

## Запуск

### Вариант 1: Локальный HTTPS сервер (рекомендуется)

```bash
# Установить http-server с SSL
npm install -g http-server

# Запустить с SSL
http-server -S -C cert.pem -K key.pem -p 8080
```

### Вариант 2: Python HTTPS сервер

```bash
# Создать самоподписанный сертификат
openssl req -new -x509 -keyout key.pem -out cert.pem -days 365 -nodes

# Python 3
python3 -m http.server 8080 --bind 0.0.0.0
```

### Вариант 3: Использовать ngrok или serveo для туннелинга

```bash
# С помощью ngrok
npx http-server -p 8080
ngrok http 8080
```

## Использование

1. Откройте браузер на вашем компьютере или прямо в Meta Quest 3
2. Перейдите по адресу `https://localhost:8080` (или IP адресу вашего компьютера)
3. Наденьте VR очки
4. Нажмите кнопку **"Enter VR"** в правом нижнем углу
5. Используйте отладочную панель для настройки параметров:
   - **Distance** - расстояние от пользователя (1-5 метров)
   - **Width** - ширина панели (1-5 метров)
   - **Curved View** - переключение между плоским и изогнутым видом

## Настройки по умолчанию

- **Расстояние**: 1.8 метра
- **Размер панели**: 2048×1152 px (2.5m × 1.4m в VR)
- **Режим**: Плоский

## Технические детали

### Архитектура

- **WebXR API** для VR сессии
- **WebGL** для рендеринга 3D сцены
- **Canvas 2D** для конвертации HTML в текстуру
- **SVG foreignObject** для рендеринга HTML контента

### Параметры конфигурации (app.js)

```javascript
const CONFIG = {
    distance: 1.8,      // Расстояние от пользователя (м)
    width: 2.5,         // Ширина панели (м)
    height: 1.4,        // Высота панели (м)
    pixelWidth: 2048,   // Разрешение текстуры
    pixelHeight: 1152,
    curved: false       // Плоский/изогнутый вид
};
```

## Отладка

Если VR не запускается:
- Проверьте, что используете HTTPS
- Убедитесь, что VR очки подключены и распознаются браузером
- Откройте консоль браузера (F12) для просмотра ошибок
- Проверьте права доступа к VR устройству в настройках браузера
